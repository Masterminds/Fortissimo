Fortissimo is a framework for rapidly building PHP-based web sites and
web applications.

To begin, use the fff.php command.

Requirements:
- QueryPath
- Phing
- PEAR
- PHPDocumentor
- PHPUnit

The code in the Fortissimo project is released under the LGPL v. 2.1.
